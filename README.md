# C# Gradebook (Console + JSON)

A simple console application that stores students and grades in a local JSON file. Good for showcasing basic C#, file I/O, validation, and LINQ.

## Features
- Add/list students and grades
- Validation for inputs
- Persists data to JSON (gradebook.json)

## How to run

**Prerequisite:** Install .NET SDK (8.0 recommended)

1. Open a terminal in this folder
2. Run:
   - `dotnet run`

The app will create/update `gradebook.json` in the same folder.
